# 🐵 Mimicando

> **O jogo de mímica, desenho e criatividade para animar qualquer reunião!**

Mimicando é um jogo de festa para grupos, rodando direto no navegador — sem instalação, sem cadastro, só diversão. Uma equipe atua, a outra adivinha: seja por mímica, desenho ou descrição, cada rodada é uma nova chance de rir (e de se enrolar).

---

## Como funciona

O jogo é por **times**. A cada rodada, um jogador escolhe a palavra secreta e usa o modo ativo para que o time adivinhe no menor tempo possível. Quem bater a pontuação-alvo primeiro vence.

### Modos de jogo

| Modo | Como jogar |
|------|------------|
| 🎭 **Mímica** | Atue sem falar nem apontar |
| ✏️ **Desenho** | Desenhe para o time adivinhar |
| 💬 **Descreva** | Explique sem dizer a palavra |
| ⚡ **Turbo** | Rodadas relâmpago de 30 segundos |

### Baralhos de palavras

Quatro baralhos gratuitos para começar e quatro extras para quem quer ir além:

**Grátis:** 🐵 Animais · 🎬 Filmes · 👩‍🚀 Profissões · 🤸 Ações

**Extras:** 🪑 Objetos · ⭐ Famosos · 🍕 Comidas · 🗺️ Lugares

---

## Configurações da partida

- ⏱ Tempo por rodada ajustável
- 🏆 Pontuação-alvo configurável
- 🔊 Efeitos sonoros e vibração (haptics)
- ❌ Opção de penalidade para palavras puladas
